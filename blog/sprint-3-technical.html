An analogy to describe the differences between HTML and CSS

HTML is like a builder who has the tools and materials to put a house together. CSS can be thought of as an architect who designs the house, and/or an interior designer who determines the colour schemes.

Explain control flow and loops using an example process from everyday life, for example 'waking up' or 'brushing your teeth'. (But not those ones).

A control flow is the sequence in which the code is read through.

Loops tell a code segment to repeat a command until a specific point is reached.

Describe what the DOM is and an example of how you might interact with it.

The DOM (Document Object Model) is a standardised coding interface that categorises HTML and XML pages into tree structures and nodes It allows for rapid access and modification with languages such as JavaScript, by way of developer tools including Visual Studio Code.

Explain the difference between accessing data from arrays and objects.

An array stores multiple data values within a single variable. An object builds on arrays by allowing different properties.

Explain what functions are and why they are useful.

Functions take one or more values as input, and return a calculated result. They are useful when code has to be frequently reused.
